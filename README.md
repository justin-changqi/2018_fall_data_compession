## Hw#1 Huffman Code
### System requirement
- CMake 2.8+
- C++ 11

### Build and Execute
```
git clone https://github.com/justin-changqi/2018_fall_data_compression.git
cd hw#1_entropy_huffman_golomb/build/
# Build
cmake ..
make
# Execute
./huffman_code
```
